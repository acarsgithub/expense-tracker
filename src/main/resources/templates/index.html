Welcome! Read the following directions to understand how to use the expense tracker application!



  • /login ---- will allow you to enter your username and password to login



  • /add-new-account/{username} ---- will allow you to add a new account to your expense manager



  • /all-users ---- will allow the admin to see all users in the database and their net worth



  • /transaction-history/{username} ---- will show all transactions for a particular user



  • /total-net-worth/{username} ---- will display a user's total net worth



  • /modify-account/{username} ---- will allow the user to deposit or withdraw money from a specific account
